# Taller Práctico: Introducción a JavaScript
## Parte 1: Explorando JavaScript
**1.1** Lenguaje de programacion para hacer las paginas web interactivas y dinamicas, mientras que Html estructura el contenido y CSS se encarga de su apariencia visual, JavaScript es el motor que le da vida, que sea funcional.

**1.2**  Caracteristicas Principales:

- Lenguaje interpretado.
- Es flexible y dinamico
- Multiplataforma
- Paradigma de programacion orientado a objetos
